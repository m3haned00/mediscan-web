# 💊 MediScan AI: Advanced Drug Interaction Analyzer

**MediScan AI** is a professional rapid-prototype web application built with **React** and powered by **LLMs (Llama 3.1 via Groq)**. It helps users identify dangerous drug-to-drug interactions with a modern UI.
